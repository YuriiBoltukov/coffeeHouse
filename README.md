# Coffee Shop Website

Проект кофейни на TypeScript с интеграцией API

## Технологии

- TypeScript
- Webpack
- ESLint
- HTML/CSS

## Установка

```bash
npm install
```

## Запуск

```bash
# Development
npm run dev

# Production build
npm run build
```

## Структура проекта

- `src/` - исходный код
- `dist/` - собранный проект
- `.github/workflows/` - GitHub Actions для деплоя

## Деплой

Деплой настроен на GitHub Pages через GitHub Actions
